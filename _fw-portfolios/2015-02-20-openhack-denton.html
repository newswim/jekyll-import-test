OpenHack Denton

SUMMARY

OpenHack is a meetup with a simple purpose: Code together, on anything. It's a great way to get programmers, developers, and students of any experience level together to code.

If you're learning how to code, looking for an open source project to work on, or just needing some technical advice on a problem you're having - this is the meetup for you.

Expect to meet amazing cofounders, hackers, and excited newbies as we all learn to build more better faster.
